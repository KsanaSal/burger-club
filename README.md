# burger-club

The site was created with Tailwind styles

Connection steps Tailwind:

1. Install Tailwind CSS

Install tailwindcss and @tailwindcss/cli via npm - [npm install tailwindcss @tailwindcss/cli]

2. Import Tailwind in your CSS

Add the @import "tailwindcss"; import to your main CSS file - [@import "tailwindcss";]

3. Start the Tailwind CLI build process

Run the CLI tool to scan your source files for classes and build your CSS - [npx @tailwindcss/cli -i ./src/style.css -o ./src/output.css --watch]

4. Start using Tailwind in your HTML

Add your compiled CSS file to the <head> and start using Tailwindâ€™s utility classes to style your content - [<link href="./output.css" rel="stylesheet">]
